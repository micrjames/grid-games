    About the Games

    The games presented here are built on a grid system using plain JavaScript, combined with html and css, in this case, transpiled from scss files.

    Each game works a little differently, so the rules and the information necessary to play them is made available when the game is selected.

    However, all of these games should be familiar to just about everyone, so it shouldn't be too tough to pick it up.
